# WorkFlow - Smart Work Manager

**업무의 시작부터 분석까지, 하나의 앱으로.**

WorkFlow는 모든 기업과 팀이 무료로 사용할 수 있는 스마트 업무 관리 앱입니다. 업무일지 작성, 출근 관리, 팀 협업, AI 인사이트 분석까지 — 복잡한 업무 환경을 하나의 앱으로 깔끔하게 정리해드립니다.

---

## 이런 분들에게 추천합니다

- 팀원들의 업무 현황을 한눈에 파악하고 싶은 **관리자**
- 매일 반복되는 업무 보고를 간편하게 처리하고 싶은 **직장인**
- 여러 팀을 운영하면서 업무 공유 범위를 조절하고 싶은 **중소기업 대표**
- 음성으로 빠르게 업무일지를 남기고 싶은 **외근 담당자**

---

## 주요 기능 소개

### 1. 음성 녹음 업무일지
키보드 입력이 번거로우신가요? **음성 녹음 버튼**을 누르고 말하면 자동으로 텍스트로 변환됩니다. 변환된 내용은 **6하원칙(누가, 언제, 어디서, 무엇을, 어떻게, 왜)**에 따라 자동 분류되어 체계적인 업무일지가 완성됩니다.

### 2. 출근 체크 시스템
매일 아침 앱을 열면 **출근 체크 팝업**이 자동으로 나타납니다.
- **내근 / 외근** 중 선택
- 외근 시 한 줄 업무 요약 입력 가능
- 전체 인원이 체크를 완료할 때까지 팝업이 반복 표시
- **출근 현황판**에서 전 직원의 내근/외근 상태를 실시간 확인

### 3. 멀티 기업 & 팀 관리
하나의 앱에서 여러 회사가 독립적으로 운영됩니다.
- **회사 등록**: 새 회사를 만들면 고유한 **6자리 초대 코드**가 발급됩니다
- **팀원 초대**: 초대 코드를 공유하면 누구나 회원가입 시 해당 회사에 참여 가능
- **팀 생성**: 회사 내에서 팀을 만들어 조직을 구성합니다
- **데이터 격리**: 회사 간 데이터는 완전히 분리되어 보안이 보장됩니다
- **팀별 공유 설정**: 팀 단위로 업무 공유 ON/OFF를 자유롭게 설정

### 4. AI 귀납적 인사이트 분석
축적된 업무 데이터를 기반으로 **AI가 귀납적 추론 분석**을 제공합니다.
- **분석 범위 선택**: 개인 / 회사 전체 / 팀별
- **기간 필터**: 1주일, 2주일, 1개월 또는 직접 설정
- **카테고리 필터**: 특정 업무 유형만 분석 가능
- **제공되는 분석 결과**:
  - 업무 통계 (총 건수, 일 평균, 활성 카테고리 수)
  - 일별 업무 추세 차트
  - 카테고리 분포 시각화
  - **긍정적 전망** — 발전 가능성 및 성장 예측
  - **리스크 분석** — 잠재적 문제점 조기 감지
  - **추천사항** — 우선순위별 개선 방향 제시

### 5. 업무 캘린더 뷰
달력 형태로 업무일지를 한눈에 조회합니다. 날짜별로 어떤 업무를 했는지 빠르게 확인할 수 있습니다.

### 6. 자동 보고서 생성
- **주간 보고서**: 한 주간의 업무를 자동으로 요약하여 보고서를 생성합니다
- **월간 요약 리포트**: 한 달간의 성과를 정리한 리포트를 제공합니다
- **인수인계 문서**: 담당자 변경 시 업무 인수인계 문서를 자동으로 생성합니다

### 7. 팀 실적 대시보드 (관리자)
관리자는 팀 전체의 업무 실적을 대시보드에서 한눈에 파악할 수 있습니다. 누가 얼마나 활발하게 업무를 수행하고 있는지 직관적으로 확인됩니다.

### 8. 편의 기능
- **즐겨찾기(북마크)**: 중요한 업무일지를 북마크하여 빠르게 접근
- **타임라인 활동 피드**: 최근 활동 내역을 시간순으로 확인
- **빠른 메모**: 간단한 메모를 즉시 기록

## 사용 방법

### Step 1. 회원가입
앱에 접속하여 **회원가입** 버튼을 누릅니다.
1. 이름, 연락처(010-XXXX-XXXX), 비밀번호를 입력합니다
2. 가입 유형을 선택합니다:
   - **회사 참여** — 기존 회사의 6자리 초대 코드를 입력하고, 팀과 직책을 선택합니다
   - **새 회사 등록** — 회사명을 입력하면 새 회사가 생성되고 초대 코드가 발급됩니다
   - **개인 사용** — 회사 없이 개인 업무 관리용으로 바로 시작합니다

### Step 2. 출근 체크
매일 앱을 열면 출근 체크 팝업이 나타납니다. **내근** 또는 **외근**을 선택하고, 외근이라면 간단한 업무 요약을 입력합니다.

### Step 3. 업무일지 작성
하단 메뉴에서 **업무일지** 탭을 선택합니다.
- **텍스트 입력**: 직접 타이핑하여 업무 내용을 작성합니다
- **음성 입력**: 마이크 버튼을 눌러 음성으로 기록하면 자동 텍스트 변환 및 6하원칙 분류가 됩니다

### Step 4. 인사이트 확인
**인사이트** 탭에서 스마트 인사이트 분석을 실행합니다.
- 분석 범위(개인/회사/팀)와 기간을 선택하고 **분석 시작** 버튼을 누르면 AI가 업무 패턴을 분석하여 전망과 추천사항을 제공합니다.

### Step 5. 보고서 활용
- 주간/월간 보고서가 자동 생성되므로 별도 작성 없이 바로 활용 가능합니다
- 인수인계가 필요할 때 자동 생성 기능을 이용하면 빠르고 정확합니다

## 보안 정책

- 회사 간 데이터는 **완전히 격리**되어 타 기업의 정보를 볼 수 없습니다
- 시스템 관리자는 서비스 운영을 위해 모든 데이터에 대한 열람 권한을 가집니다
- 팀 내 업무 공유 여부는 관리자가 자유롭게 설정할 수 있습니다
