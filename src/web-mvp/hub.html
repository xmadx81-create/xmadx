게임 규칙

개요

당신은 백십자재단 산하 혈연센터의 관리자입니다. 카르테인 가문의 뱀파이어들이 센터에 침투하고 있습니다. 팀을 이끌고 격자 맵에서 전투하여 센터를 지키세요.

전투 진행

  1. 아군 턴 — 모든 아군 유닛을 이동/공격/스킬 사용
  2. 적 턴 — 적 AI가 자동으로 이동·공격
  3. 반복하여 승리 조건 달성

유닛 스탯

  • HP — 체력. 0이 되면 전사
  • ATK — 공격력. 높을수록 큰 데미지
  • DEF — 방어력. 받는 데미지 감소
  • MOV — 이동력. 한 턴에 이동 가능 칸 수
  • RNG — 사거리. 1=근접, 2=원거리

전투 시스템

  • 데미지 = ATK - DEF + 랜덤(0~3)
  • 15% 확률로 크리티컬 (1.5배 데미지)
  • 공격받은 유닛이 사거리 안이면 반격 (70% 위력)

팩션 상성

  • 🏥 혈연센터 → 카르테인 상대 시 DEF +2
  • 🧛 카르테인 → 비소속 상대 시 ATK +3
  • 🤝 비소속 → 혈연센터 상대 시 선제 반격

촉(觸) vs 혈(血) 스킬

각 캐릭터는 고유 특수 스킬을 보유합니다. 사용 후 3턴 쿨다운.

촉(觸) — 인간의 감각

  • 👁 예감 — 아군 DEF 버프 (↔혈향)
  • ⚡ 직감 — 적 직접 데미지 (↔혈압)
  • 🔗 감응 — 아군 회복 + 상태이상 해제 (↔혈유)
  • 🩸 혈각 — 적 ATK 디버프 (↔혈식)
  • 🔥 투지 — 자신 ATK/DEF 버프 (↔혈기)
  • 💛 공감 — 광역 회복 + 상태이상 해제 (↔혈맹)

혈(血) — 뱀파이어의 힘

  • 👊 혈압 — 적 단일 고데미지
  • 🌬 혈향 — 적 독/출혈 부여
  • ⛓ 혈맹 — 아군 ATK 버프
  • 🌑 혈식 — 적 DEF 관통 데미지
  • 💀 혈기 — 자가 회복 + 출혈 부여
  • 🫦 혈유 — 적 둔화 + 데미지

전투 팁

  • 협공 — 방어자 인접 아군 1명당 ATK +2 추가
  • 지형 활용 — 숲(DEF+1), 묘지(ATK+2), 온천(매턴 HP+5), 늪(DEF-1)
  • 위험 범위 — 🔴 버튼으로 적의 공격 가능 범위 확인
  • 킬 예보 — 공격 전 "확정 처치" / "반격 위험" 표시 확인

상태이상

  • 🟢 독 — 매턴 지속 데미지
  • 🩸 출혈 — 매턴 지속 데미지
  • 💫 기절 — 1턴 행동 불가
  • 🐌 둔화 — 이동력 감소

장비 & 유물

유닛에 자동 배분되는 장비와 유물이 스탯을 강화합니다.

무기 (ATK/관통/크리)

  • 🗡 채혈침 — ATK +2
  • 🗡 보안봉 — ATK +4, CRT +5%
  • 🗡 수술용 메스 — ATK +6, PEN +2
  • 🗡 카르테인 장검 — ATK +8, PEN +3, CRT +10%
  • 🗡 듀크의 의검 — ATK +12, PEN +5, CRT +15%

방어구 (DEF/HP/회피)

  • 🛡 실험복 — DEF +2, HP +5
  • 🛡 방호복 — DEF +4, HP +10
  • 🛡 혈맹 갑주 — DEF +6, HP +20, EVA +5%
  • 🛡 고대 혈족 갑주 — DEF +10, HP +30, EVA +10%

장신구 (특수)

  • 👓 야간 투시경 — RNG +1
  • 👟 경량 부츠 — MOV +1
  • 💍 날카로운 부적 — CRT +15%, ATK +3
  • 💍 듀크의 인장 — ATK +5, CRT +15%, EVA +10%

유물 (조건부 효과)

  • 🔴 첫날의 각오 — HP 100%일 때 ATK +20%
  • 🟡 생존자의 의지 — HP 25% 이하일 때 DEF +50%, ATK +30%
  • 🩸 혈연의 계약 — 공격 시 데미지의 10% 흡수
  • 👓 센터장의 안경 — 아군 전체 DEF +2
  • 🍷 듀크의 와인잔 — 적 처치 시 HP 30% 회복

MBTI 시너지

각 캐릭터는 고유 MBTI를 보유합니다. 같은 팀에 배치 시 MBTI 궁합에 따라 팀 시너지 배율이 결정됩니다.

  • SS — 최고 궁합 (×1.5)
  • S/A — 우수 (×1.2~1.3)
  • B/C — 보통 (×1.0~1.1)
  • 시크릿 콤보: 특정 조합 시 추가 보너스

궁극기

일부 캐릭터는 레벨 해금 궁극기를 보유합니다. 높은 MP를 소모하지만 강력한 광역/버프 효과를 발휘합니다.

패시브 트리

각 역할별 3단계 패시브가 존재합니다. 레벨이 오르면 자동 해금되어 스탯이 영구 상승합니다.

하드 모드

일반 클리어한 스테이지에서 해금됩니다. 적 레벨 ×2, ATK/DEF +30%. 보상으로 전설 카드를 획득할 수 있습니다.

카드 모집

전투 보상으로 받는 🎫 모집권으로 랜덤 카드를 획득할 수 있습니다. 확률: 커먼 60%, 언커먼 28%, 레어 10%, 전설 2%.

카드 합성

도감에서 보유 카드를 합성하여 레벨을 올릴 수 있습니다. 중복 카드 3~5장으로 1레벨 상승.

날씨

일부 스테이지와 무한의 탑에서 날씨가 전투에 영향을 줍니다.

  • ☀️ 맑음 — 효과 없음
  • 🌧️ 비 — 원거리 ATK-2, 전체 EVA+5%
  • 🌫️ 안개 — 사거리 감소, 원거리 ATK-3
  • ❄️ 눈보라 — 전체 MOV-1, 빙판 EVA+10%
  • 🌑 혈월 — 카르테인 ATK+3, 회복 -30%
  • ⛈️ 폭풍 — 전체 CRT+10%, 반격+20%

무한의 탑

끝없는 웨이브에 도전하는 모드. 층이 오를수록 적이 강해지고 날씨가 변합니다. 5층마다 증원이 출현합니다.

승리 / 패배

  • 승리: 모든 적 제압 (또는 생존 조건 달성)
  • 패배: 모든 아군 전사
  • 보상: ⭐⭐⭐ = 레어 카드, ⭐⭐ = 언커먼, ⭐ = 커먼
