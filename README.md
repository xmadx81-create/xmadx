# 석유사업본부 업무공유 시스템

석유사업본부의 업무 편의성 향상을 위한 모바일 퍼스트 웹 애플리케이션

## 주요 기능

- **업무일지** - 일일/주간 보고서 작성 (내근/외근/출장)
- **육하원칙(5W1H)** - 누가, 언제, 어디서, 무엇을, 어떻게, 왜에 따른 체계적 업무 기록
- **주간계획** - 요일별 업무 계획 수립
- **가맹관리** - 지역 가맹점 관리 및 신규 가맹 영업 추적
- **결재라인** - 작성자 > 1차 결재 > 2차 결재 워크플로우
- **수신 지정** - 업무보고 수신자 지정
- **반복업무 템플릿** - 자주 반복되는 업무를 템플릿으로 저장/불러오기
- **자동완성** - 이전 입력 기반 자동 추천 (선택 가능)
- **업무표 자동생성** - 개인별 업무 기록을 기간별 업무표로 자동 종합
- **개인업무 매뉴얼 자동생성** - 축적된 업무 데이터 기반 매뉴얼 자동 생성
- **방문기록** - 가맹점 방문 이력 및 후속 조치 관리

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속

## 기술 스택

- **Backend**: Node.js + Express
- **Database**: SQLite (better-sqlite3)
- **Frontend**: HTML5 + CSS3 + Vanilla JS (모바일 퍼스트)
- **PWA**: 스마트폰 홈화면 추가 지원
